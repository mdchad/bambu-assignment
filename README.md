# Job Search App

This is a Bambu assignment for the full stack position. The assignment is to create
a job search application 

## Getting Started

You can accessed here: [Deployed App](https://d1l0buvv1sm9ma.cloudfront.net/)

or can start locally: 

```
npm i
npm run dev   
```
